Agent Baseline Configuration

  • Rules/Instructions files — Project-level context (CLAUDE.md, .cursorrules, .clinerules)
  • Permission modes — Control what the agent can do (read, write, execute)
  • Approved tools — Allowlist of CLI tools and commands
  • Model selection — Choose the LLM backend (local, cloud, or hybrid)
Before you extend agents with MCP, you need a solid baseline configuration.
