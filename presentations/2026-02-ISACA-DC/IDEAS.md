# Ideas for this presentation

## ISACA-DC Feb 19, 2026

- Co-presented with Brian Pasquini (Pitt / bluefoxlabs.ai)
- Audience: IT audit, security, and governance professionals
- Core theme: 0-3 levels of agentic coding, good/bad/ugly, live coding demo
- Based on "AI Coding In Practice" deck

### Structure
- Brian: AI's Role in Problem Solving (lifecycle)
- Jon: The Good, Bad, and Ugly of AI coding
- Agentic Coding Landscape (levels 0-3)
- Live coding demo with Gastown / Claude Code
- Zenable guardrails tie-in

### Levels of Agentic Coding
- Level 0: Chat-based (ChatGPT, Claude, Gemini)
- Level 1: Editor-integrated (Copilot, Cursor, Windsurf)
- Level 2: CLI agents (Claude Code, Codex CLI, Aider)
- Level 3: Orchestrators (Gastown, OpenHands, Devon)
