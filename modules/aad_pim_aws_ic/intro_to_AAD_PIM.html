Initially meant for Azure Roles Now they have "PIM for Groups" renamed in January 2023 from as PAGs (Privileged Access Groups). NEED to set it as role-enabled on group creation (recreating groups). You can set a group to make someone eligible to activate a PAG. - Consider Identity Governance and Access Review to do this.
